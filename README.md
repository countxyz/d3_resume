# D3 Resume

## About

My resume in D3 using coffeescript within the Sinatra framework.

## Start Hacking

```bash
git clone git@github.com:countxyz/d3_resume.git
cd d3_resume
bundle install
bundle exec shotgun
```
Navigate to http://127.0.0.1:9393
